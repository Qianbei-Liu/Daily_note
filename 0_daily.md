#### 终端分屏操作

+ 使用工具

  1，输入命令tmux使用工具

  2，上下分屏：ctrl + b 再按 “

  3，左右分屏：ctrl + b 再按 %

  4，切换屏幕：ctrl + b 再按o

  5，关闭一个终端：ctrl + b 再按x

  6，上下分屏与左右分屏切换： ctrl + b 再按空格键
